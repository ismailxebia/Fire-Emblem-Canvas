# Fire Emblem Canvas - React Version

This project has been migrated to React + TypeScript + Vite.

## Structure
- `src/components/Game.tsx`: The main Game component that initializes the Canvas and Game Engine.
- `src/game/`: The original game engine logic (JS files), adapted for React.
- `src/index.css`: Styles migrated from the original CSS.

## Running
1. `npm install`
2. `npm run dev`

## Building
`npm run build`

## Legacy Code
The original code is preserved in `_legacy/` for reference.
